# Recon-Tool

### PreCondition
* Website (ie google.com) says HTTP/1.1 200 OK

### Passive Recon
* Whois
* NSlookup
* Wayback Enum
* TheHarvester

### Active Recon
* Nikto
* nmap enum
* Robots.txt

### Manual Recon
* ViewDNS.info (website)
* Pipl (website)
* Netcraft (website)
* Maltego (all-in-one)
* Sn1per (all-in-one)
